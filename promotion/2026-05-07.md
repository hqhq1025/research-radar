# 推广策略观察 2026-05-07

## ARIS 的「AI 自己做科研」叙事

ARIS 拿到 68 upvotes 的核心策略是**量化叙事**——不说「AI 能做研究」，而是说「论文评分从 5.0 到 7.5」「自主发起 20+ GPU 实验」。这种具体数字让抽象能力变得可信。加上「开源」标签降低信任门槛，形成了强传播力。

## OpenSeeker-v2 的「反共识」定位

「不用 RL 也能 SOTA」本质上是一种反共识叙事，在技术圈极易引发讨论。当所有人都在做复杂 RL pipeline 时，你说 SFT 就够了，这种「less is more」的故事自带争议性和转发动力。

## Agent 框架的生态卡位

GitHub 上 Composio 和 Craft-Agents 快速涨星，核心策略是**生态集成**——不做单一模型，而是做连接层。列出所有支持的平台（Anthropic、OpenAI、LangChain...）本身就是一种 SEO 和社交传播策略，每个关键词都是流量入口。
